Downloads & Code

  • Python (docs) 
    pip install smk_python_sdk
    easy_install smk_python_sdk

    or build yourself smk_python_sdk-0.3.2.tar.gz 

    python setup.py build install

  • Erlang - add the following dependancy to your rebar.config and checkout the README on GitHub 

    {smk, "0.3.2", {git, "https://github.com/smarkets/smk_erlang_sdk.git", {tag, "v0.3.2"}}}

  • .piqi and .proto specs and static docs - smk_api_docs-0.5.2.tar.gz

  • Smarkets on GitHub

    • https://github.com/smarkets/smk_erlang_sdk - Erlang smk application

    • https://github.com/smarkets/smk_python_sdk - Python smk library

    • https://github.com/smarkets/IronSmarkets - C# library

    • https://github.com/smarkets/smk_api_docs - Source for the API docs

    • https://github.com/mokele/smk_java_debug - Java connection/login example
